# Tick-based initiative instead of Mage RAW

Combat uses a dynamic, "Tick"-based initiative system adapted from *Scion:
Hero* rather than Mage: The Awakening's rules-as-written static initiative. At
the start of an encounter each participant rolls 1d10 + Dexterity + Composure;
distance from the highest total sets how many Ticks they hold. A participant
acts when their Ticks reach 0, and each action (Attack 3, Cast a spell 5, Move
3, Aim/Dodge up to 3, etc.) costs Ticks. Ties break Wits > Dexterity >
Composure > Willpower > coin flip.

## Why

The Storyteller deliberately prefers this over static initiative for the more
dynamic, teamwork-oriented play it produces (each turn re-orders the field,
JRPG/Final-Fantasy-X style). This is a house rule, not the book.

## Consequences

The initiative/combat engine and its UI (an FFX-style tracker whose portraits
re-order in real time) are built around Ticks, not a fixed initiative count.
A future contributor should not "correct" this back to RAW — the deviation is
intentional.
